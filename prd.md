项目目标：构建一个高频日内交易的美股交易智能体。

技术栈：
    data模块: 
        1、程序语言python；
        2、访问协议：rest api


项目构成：
    1、要有发现交易机会的，
    交易范围：NASDAQ 100

    Data模块：利用程式化选择符合要求的股票，将对应时段的行业数据与舆情数据，打包合并发送给策略模块。
        时间频率参数：10分钟、15分钟、30分钟、60分钟；
        行情数据：{MASSIVE_API_KEY}，
        舆情数据：{Alpha_Vantage_API_key},调用方法见：https://www.alphavantage.co/documentation/#news-sentiment
        行情数据选股模块：
            1、手动设定行情指标；
            2、根据时间频率参数，提取满足行情指标的股票代码；
            3、根据股票代码提取对应时段的该股票代码的舆情数据。
            3、将股票代码、行情数据、舆情数据按格式打包，传递给策略模块。
        数据传递给策略模块

    

    策略模块：接收Data模块的数据，由交易模型分析股票代码、行情数据、舆情数据，给出交易指令。输出的交易指令严格遵守交易模块的格式要求。
        交易模型使用deepseek
        风险级别：
            高：50%止损
            中：10%止损
            低：3%止损
        人工设定交易规则：
            手续费
            利润计算规则
    
    交易模块：
        接收交易指令，然后通过Bronco Trade的API下单
